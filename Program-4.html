Introduction to HTML

What is HTML?




HTML stands for Hyper Text Markup Language. The first version of HTML was written by Tim Berners-Lee in 1993.
HTML is Markup language, and is different from coding.The pages created on HTML are viewed on World Wide Web (www).
It can be easily created by usinf Notepad editor, and is saved by the extension .html.
It is easy to learn and use, and is widely used. It is supported by all browsers.
It is used to create Webpages, Webdocument, Internet Navigation, Client Side Stroage and much more.
